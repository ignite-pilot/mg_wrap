## 3. 보관 신청 (Storage Application)

### 3.1 공간형 자산보관
- [ ] "공간형 자산보관" 버튼 선택 가능
- [ ] 평수 입력 필드 표시 확인
- [ ] 평수 입력 (1평 이상)
- [ ] 보관 개월수 선택 (1개월, 3개월, 6개월)
- [ ] 견적 조회 버튼 클릭
- [ ] 견적 금액 정확성 확인
  - [ ] 1개월: 평수 × 60,000원
  - [ ] 3개월: 평수 × 266,666원 (반올림)
  - [ ] 6개월: 평수 × 500,000원
- [ ] "이용 요금" 섹션 표시 확인
- [ ] 최종 신청 버튼 클릭
- [ ] 신청 성공 메시지 확인
- [ ] 신청 내역 테이블에 추가 확인

### 3.2 BOX형 자산보관
- [ ] "BOX형 자산보관" 버튼 선택 가능
- [ ] BOX 수량 입력 필드 표시 확인
- [ ] BOX 수량 입력 (1개 이상)
- [ ] 보관 개월수 선택
- [ ] 견적 조회 버튼 클릭
- [ ] 견적 금액 정확성 확인
  - [ ] 1개월: BOX 수량 × 30,000원
  - [ ] 3개월: BOX 수량 × 80,000원
  - [ ] 6개월: BOX 수량 × 150,000원
- [ ] 최종 신청 버튼 클릭
- [ ] 신청 성공 메시지 확인
- [ ] 신청 내역 테이블에 추가 확인

### 3.3 입력 검증
- [ ] 평수/BOX 수량 미입력 시 에러 메시지
- [ ] 평수/BOX 수량 0 이하 입력 시 에러 메시지
- [ ] 견적 조회 전 최종 신청 시도 시 에러 메시지
- [ ] 에러 메시지와 버튼 간 적절한 간격 확인

### 3.4 신청 내역
- [ ] 신청 내역 테이블 표시 확인
- [ ] 테이블 컬럼 확인 (유형, 단위, 개월수, 견적 금액, 상태, 신청일)
- [ ] 신청 내역이 없을 때 "신청 내역이 없습니다" 메시지 표시
- [ ] 여러 신청 내역 표시 확인

- [ ] 잘못된 날짜 형식 시 에러 메시지
- [ ] 잘못된 자산 분류 시 에러 메시지

### 4.5 자산 목록
